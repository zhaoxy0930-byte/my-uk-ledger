# Ledger UK

一个本地优先的英国个人记账分析原型。

## 使用方式

直接在浏览器打开 `index.html`。也可以点击页面右上角的“示例数据”快速查看效果。

如果浏览器限制本地文件访问，可以启动本地服务器：

```bash
node server.mjs
```

然后访问 `http://127.0.0.1:8765`。

PDF 导入建议使用本地服务器方式打开，因为浏览器直接打开 `index.html` 时可能会限制 PDF 解析模块。

## 当前支持

- 上传 CSV / TXT / 文字型 PDF 银行账单 / JPG、PNG、WebP 截图
- 新导入批次支持一键撤销本次导入
- 自动识别常见字段：日期、描述、商户、收入、支出、余额
- 从银行导出的 PDF statement 中抽取文字交易行
- 用本地 OCR 引擎识别手机银行截图，包括按日期分组的交易列表
- 支持 Lloyds 表格式截图，按 Type / Money In / Money Out / Balance 判断收入支出
- 英国常见银行格式的宽松识别：Monzo、Starling、Barclays、HSBC、Lloyds、NatWest、Santander、Revolut、Chase UK
- 自动分类与手动修正
- 在交易表中直接编辑日期、商户、描述、金额和类目
- 一键把错误识别的支出标记为退款
- 交易列表支持分页、每页数量、待处理、未分类、待确认、退款筛选
- 可新增、编辑、删除自动分类规则
- 内置退款 / cashback / reimbursement 识别
- 自定义支出/收入类目
- 日、周、月、年统计
- 总览可切换上一周期 / 本期 / 下一周期，并显示当前周期与上一周期对比
- 图表支持鼠标悬停查看收入、支出、净现金流数值
- 固定工资、房租、订阅、账单识别
- 数据保存在浏览器本地 `localStorage`
